About Me

Hi, I’m Cihat — a front-end developer who values clean design, accessibility, and user experience. I started learning HTML, CSS, and JavaScript on my own, and quickly fell in love with how the web works and what it can do.

I focus on creating responsive layouts, intuitive interfaces, and performance-optimized front-end structures. I also design and build websites with WordPress, using tools like Elementor and Gutenberg to create flexible, user-friendly, and visually appealing content management experiences.

I care about writing clean, maintainable code and constantly look for ways to improve through every project I take on. My goal is to build simple, effective digital experiences that solve real user problems. I'm collaborative, curious, and always excited to turn ideas into interactive reality.
